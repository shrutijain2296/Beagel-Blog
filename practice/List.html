Block Elements

Block Elements are those which start from a new line and takes complete width of the screen

Example:

  1. p
  2. ul
  3. h1 - h6
  4. ol
  5. pre

Inline Elements

Inline Elements are those which start from the point where the previous element ended

Example:

  • a
  • small
  • br
  • object
  • time
